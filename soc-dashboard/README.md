# SOC Dashboard

React frontend for the AI-SOC Watchdog system.

## Setup

```bash
npm install
npm run dev
```

Runs on http://localhost:5173. Requires the backend running on port 5000.

## Pages

- Login - Authentication page
- AnalystDashboard - Main alert monitoring view
- TransparencyDashboard - AI decision transparency
- RAGDashboard - RAG system status and queries
- PerformanceDashboard - System performance metrics
- DebugDashboard - Development debugging tools

## Components

- Sidebar - Navigation sidebar
- ErrorBoundary - Error handling wrapper
